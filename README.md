Cerebrums
=========

何をするものか / 目指すものか
-----------------------------

* 文章・情報共有アプリケーション
	- 現時点では Markdown 文章限定かつ、共有(ネットワーク機能)は未実装
* リアルタイムMarkdownプレビュー
* ファイルシステムに依存しない文章・情報保存の仕組み
* 自前の検索エンジン (FM-Index)

動かしかた
----------

```
$ npm install
$ npm start
```
